# congestion-scraper

A minimal and niche framework for scraping congestion information from different sources and storing them as timestamped information in a databse.


## Motivation
The motivation for this project is the [constant traffic congestion](https://www.swr.de/swraktuell/baden-wuerttemberg/friedrichshafen/immer-stau-auf-b31-in-hagnau-am-bodensee-wann-kommt-umgehung-100.html) on Federal Highway B31 in and around Hagnau. The goal is to collect enough data to train a simple predictive model for the traffic congestion there.
